#Проектная работа «Пряники»

Цель работы - на основе готовых данных о продажах рассчитать **рейтинг продавцов по доходности** и **бонусы для каждого из них**.

Программа не имеет пользовательского интерфейса — все результаты отображаются в **DevTools → Console**.
---

##  Запуск проекта

1. Откройте файл `index.html` любым удобным способом
2. Откройте инструменты разработчика в браузере
3. Перейдите на вкладку **Console** — там будет выведен рейтинг продавцов и рассчитанные бонусы.

---

## Используемые технологии

- JavaScript (ES6+)

```

```
---

## Смена датасета

По умолчанию подключён первый набор данных.  
Чтобы изменить датасет, в файле `index.html` замените строку:

```html
<script src="data/dataset_1.js"></script>
```

Доступны 3 примера дата-сетов:
- `data/dataset_1.js`
- `data/dataset_2.js`
- `data/dataset_3.js`

---

## Ограничения

- Проект не содержит UI
- Нет работы с DOM
- Нет асинхронных операций
- JavaScript выполняется только в консоли
- Файлы данных и `index.html` не изменяются
